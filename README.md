# Vignesh Karunakaran's Personal Blog 🚀

![Banner](https://user-images.githubusercontent.com/26347575/133427543-2e874d92-b5ba-4204-bdfc-5a3508d903a6.png)

A modern, fast, and responsive personal blog and portfolio built with the latest web technologies.

## ✨ Features

- **🌓 Dark & Light Mode:** Seamless theme switching with system preference detection.
- **📝 Markdown-Powered Blog:** Easy content management using markdown files.
- **🎨 Modern UI:** Clean, ChatGPT-inspired typography (Inter font) and responsive design.
- **⚡ Performance:** Built on Next.js for server-side rendering and static site generation.
- **📄 Resume Integration:** Professional resume component integrated directly.
- **📱 Mobile Optimized:** Fully responsive down to 320px screens.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (Version 16/15)
- **Library:** [React](https://reactjs.org/) (Version 19)
- **Styling:** Vanilla CSS with Modern CSS Variables
- **Theme Management:** [next-themes](https://github.com/pacocoursey/next-themes)
- **Markdown Parsing:** [marked](https://github.com/markedjs/marked) & [gray-matter](https://github.com/jonschlinkert/gray-matter)

## 🚀 Getting Started

### Prerequisites

- **Node.js:** `>=20.0.0`
- **npm:** Recommended latest stable version

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vignesh-karunakaran/my-blog.git
   cd my-blog
   ```

2. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) (or your configured port) with your browser to see the result.

## 📦 Build & Deployment

To build the project for production:

```bash
npm run build
```

To start the production server:

```bash
npm run start
```
